# **App Name**: CampusConnect

## Core Features:

- User Registration: Allows users (medical professionals) to register with their details such as name, ID, city, specialty, and email, while also accepting terms for data processing. Data is synchronized in real time.
- Admin Panel: Provides an admin interface to manage user registrations, event schedules, speakers, and Q&A. Data is synchronized in real time.
- User Panel: Presents a user-friendly interface for medical professionals to view the event agenda, speaker details, and submit questions. Data is synchronized in real time.
- Dynamic Agenda Display: Displays the event agenda, including dates, times, topics, and speakers. Data is synchronized in real time.
- Speaker Information: Shows speaker profiles including name, specialization, and potentially a short bio. Data is synchronized in real time.
- Q&A Management: Enables users to submit questions to speakers, and allows admins to manage and display questions. Data is synchronized in real time.
